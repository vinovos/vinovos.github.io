碰见问题

在执行 hexo deploy 后,出现 error deployer not found:git
默认情况下，在哪个文件夹下运行npm，npm就在当前目录创建一个文件夹node_modules,
然后将要安装的程序安装到文件夹node_modules里面，所以必须在hexo目录下执行
npm install hexo-deployer-git --save
